This is a performance optimization framework. This framwork is still under development. Framework requires python 2.7.10 - 2.7.14 version and pip.


Installation 
In order to install the framework use "python Initialize.py"


Framework
1. Dependency manager
	This framework contains a python dependency manager.
	1. Install a new dependency "python Handler.py insatll [depName] [specific version]" ex: "python Handler.py insatll ompp 1.0.0
	2. Uninstall a dependency "python Handler.py uninsatll [depName] [specific version]" ex: "python Handler.py uninsatll ompp 1.0.0
	3. Update the entire dependenies after remote dependency adding( ex: git pull) "python Handler.py updateAll"
	
	Please note that both || depname - Module name || and ||specific version - standard version || 

2. OmpP profiler
	OmpP profiler is a openmp related profiling tool. ompP repository contains API documentation.
